# Local AI macOS Reference 0.3.0

This release makes the complete interface available in English and German with an immediate in-app language switch. App chrome, runtime state, errors, accessibility text, and microphone permission copy follow the selected language, while chat messages and the editable system prompt remain untouched. Sparkle now provides manual update checks and optional automatic checks through a public release-only feed secured by an app-specific EdDSA key. Published universal archives retain the local inference bridge in optimized builds and are Developer-ID signed, notarized, and stapled before entering the appcast. Development runs end to end on the Mac Studio through a safe one-way sync runner that regenerates with XcodeGen, keeps long work awake, validates the complete result bundle, verifies the app signature, and stages artifacts without installing them locally.
